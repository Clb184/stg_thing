Types:
-Config -> Configuration that game retrieves at start
-ScoreTable -> Score table compiling information about past runs
{
	

}

-ScoreInfo -> Info includes Name, Score, Date, ShotName
Retrieved from following JSON:
{
	"name" : "",
	"score" : 00000,
	"date" : "01/01/2026",
	"shotname" : "Shot sample"
}

-MusicEntry -> Music Entry fed to the Music Control, composed by:
Retrieved from following JSON:
{
	"path" : "", // Path of music file, mandatory, if failed to add, no music is played
	"loop" : true, // True by default
	"beginloop" : "0", // Start of track in samples by default
	"endloop" : "20000" // End of track in samples by default
}

-SoundEntry -> Sound Entry fed to the Sound Control, composed by:
{
	"path" : "sound.ogg", // 
	"nbuffers" : 2 // Number of buffers, default is 1
}
